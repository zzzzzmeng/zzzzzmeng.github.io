# zzzzzmeng.github.io

Personal technical blog powered by GitHub Pages.

## Publish

1. Create a public repository named `zzzzzmeng.github.io` on GitHub.
2. Push this local repository:

```powershell
cd D:\WorkSpace\Code\zzzzzmeng.github.io
git remote add origin https://github.com/zzzzzmeng/zzzzzmeng.github.io.git
git branch -M main
git push -u origin main
```

3. Open `https://zzzzzmeng.github.io` after GitHub Pages finishes deploying.

## Add a Post

Create a new HTML file under `posts/`, then add it to the post list in `index.html`.
